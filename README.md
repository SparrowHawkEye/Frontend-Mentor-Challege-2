### What I learned

Learned in this challenge. How to add solid border around button without moving the container in hover state, see snip below:

```css
.btn:hover,
.btn:active {
  color: hsla(0, 0%, 100%, 0.75);
  background: none;
  box-shadow: 0 0 0 3px hsla(0, 0%, 100%, 0.75);
}
```
